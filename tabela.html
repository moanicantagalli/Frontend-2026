Sobre a Tabela ASCII:

Os computadores representam tudo através de zeros e uns. É necessário que exista um sistema de codificação que faça a conversão de letras e caracteres para números, para o computador ter a representação binária disso.

É aí que a tabela ASCII (American Standard Code for Information Interchange) entra, um sistema de codificação que especifica os caracteres associados aos números 0 a 127, conjunto que constitui o alfabeto ASCII.

É comum usar atalhos verbais ao falar de caracteres ASCII. Por exemplo, em vez de escrever o caractere A, podemos escrever o caractere 65, já que o byte que corresponde a A na tabela ASCII representa 65 em notação binária.
